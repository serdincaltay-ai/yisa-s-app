# CELF Merkez — 12 Direktörlük

| Kodu | Direktörlük | Anahtar kelimeler |
|------|-------------|-------------------|
| **CFO** | Finans | bütçe, gelir, gider, tahsilat, maliyet |
| **CTO** | Teknoloji | sistem, kod, api, performans, hata |
| **CIO** | Bilgi Sistemleri | veri, database, entegrasyon, tablo |
| **CMO** | Pazarlama | kampanya, reklam, sosyal medya, tanıtım |
| **CHRO** | İnsan Kaynakları | personel, eğitim, performans, izin |
| **CLO** | Hukuk | sözleşme, patent, uyum, kvkk |
| **CSO_SATIS** | Satış | müşteri, sipariş, crm, satış |
| **CPO** | Ürün | şablon, tasarım, özellik, ui, sayfa |
| **CDO** | Veri | analiz, rapor, dashboard, istatistik |
| **CISO** | Bilgi Güvenliği | güvenlik, audit, erişim, şifre |
| **CCO** | Müşteri | destek, şikayet, memnuniyet, ticket |
| **CSO_STRATEJI** | Strateji | plan, hedef, büyüme, vizyon |
